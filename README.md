Project Description:

The Telecom Churn: Customer Risk Analysis Dashboard is an interactive Power BI solution developed during my Data Science Internship at Nirvaa Solutions Pvt. Ltd. The dashboard analyzes customer demographics, subscription details, service usage, and churn behavior to identify customers who are at high risk of leaving the telecom company. It helps businesses improve customer retention through data-driven insights.

Project Objective:
Analyze customer churn patterns.
Identify high-risk customers.
Monitor customer demographics and service usage.
Discover factors contributing to customer churn.
Support customer retention strategies using interactive dashboards.


Tools & Technologies:
Microsoft Power BI
Power Query
DAX
Microsoft Excel / CSV
Data Modeling
Data Visualization
